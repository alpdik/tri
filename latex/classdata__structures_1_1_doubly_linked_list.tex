class data\+\_\+structures::\+\+Doubly\+Linked\+List$<$ T $>$}
A templated doubly linked list container. 

The \doxylink{classdata__structures_1_1_doubly_linked_list}{Doubly\+Linked\+List} class stores elements in dynamically allocated nodes connected via forward and backward pointers. This structure allows efficient insertion and removal at both ends of the list and supports bidirectional iteration.
