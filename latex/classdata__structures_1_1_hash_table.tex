class data\+\_\+structures::\+\+Hash\+Table$<$ K, V $>$}
A templated hash table with separate chaining. 

The \doxylink{classdata__structures_1_1_hash_table}{Hash\+Table} class stores key–value pairs using a hash-\/based indexing mechanism. Collisions are handled via linked lists (separate chaining), and the table automatically resizes when the load factor exceeds a predefined threshold.
