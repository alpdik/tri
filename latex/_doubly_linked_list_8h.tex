\doxysubsection{Detailed Description}
A generic pointer-\/based doubly linked list container. 

Doubly\+Linked\+List class that provides a dynamic container implementation using a self made doubly linked list structure. It supports bidirectional traversal, indexed access, insertion, removal, and iterator-\/based operations.
