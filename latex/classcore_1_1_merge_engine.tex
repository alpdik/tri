Provides functionality for merging commits using a 3-\/way merge algorithm. 

The \doxylink{classcore_1_1_merge_engine}{Merge\+Engine} class compares the common base commit with the "{}ours"{} and "{}theirs"{} commits to determine the final merged state and to identify merge conflicts when they occur.
