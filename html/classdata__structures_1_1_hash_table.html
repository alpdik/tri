Detailed Description

template<typename K, typename V>
class data_structures::HashTable< K, V >

A templated hash table with separate chaining.

The HashTable class stores key–value pairs using a hash-based indexing mechanism. Collisions are handled via linked lists (separate chaining), and the table automatically resizes when the load factor exceeds a predefined threshold.

Template Parameters
KType of keys.
VType of values.
