Detailed Description

template<typename T>
class data_structures::DoublyLinkedList< T >

A templated doubly linked list container.

The DoublyLinkedList class stores elements in dynamically allocated nodes connected via forward and backward pointers. This structure allows efficient insertion and removal at both ends of the list and supports bidirectional iteration.

TType of elements stored in the list.
